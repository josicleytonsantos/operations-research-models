# Results Directory

This directory contains all outputs generated by the benchmark experiments for the 0-1 Knapsack Problem.

## Files Description

### `experiment_results.csv`
Raw experimental data generated by `run_experiments.py`.

Each row represents a solver execution for a specific instance, containing:
- instance name
- solver used
- runtime (seconds)
- objective value found
- optimality flag

This file is the base dataset used for all subsequent analyses.


### `performance_statistics.csv`
Aggregated performance metrics computed from `experiment_results.csv`.

For each solver, it includes:
- Mean runtime
- Median runtime
- Standard deviation of runtime
- Success rate (percentage of optimal solutions found)


### `runtime_boxplot.png`
Boxplot visualization of runtime distribution per solver.

This plot highlights:
- Median runtime
- Variability
- Outliers
- Comparative performance between solvers


### `statistical_test.txt`
Results of the statistical significance test comparing solver runtimes.

- If ≥ 3 solvers → Friedman test is applied.
- If 2 solvers → Wilcoxon signed-rank test is applied.

The file reports:
- Test name
- Test statistic
- p-value


### `success_rate.png`

Indicates the percentage of instances where the solver reached the optimal solution.

---

All files in this directory are automatically generated by:

```bash
python -m analysis.analyze_results
```

The directory should not be manually edited, as it is fully reproducible.
