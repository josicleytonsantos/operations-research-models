# operations-research-models

This repository contains implementations of classical Operations Research (OR) problems solved using exact optimization methods.

The goal of this project is to model, solve, and analyze well-known optimization problems using different mathematical programming tools in Python.

---

## 🎯 Objectives

- Implement classical OR problems from the literature
- Solve benchmark instances
- Compare solver performance
- Provide clear and structured mathematical formulations
- Build a practical optimization portfolio

---

## 🛠 Tools and Technologies

- Python
- PuLP
- Pyomo
- CBC Solver
- (Future: Gurobi, SCIP)

---

## 📚 Problems Implemented

| # | Problem | Type | Status |
|---|----------|------|--------|
| 1 | Knapsack Problem (0-1) | Binary Integer Programming | In Progress |

---

## 📊 Methodology

For each problem, the following steps are performed:

1. Mathematical formulation
2. Implementation in Python
3. Resolution of benchmark instances
4. Performance analysis (execution time, objective value, gap)
5. Comparison between solvers (when applicable)
