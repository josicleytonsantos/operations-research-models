# operations-research-models

This repository contains implementations of classical optimization problems using exact mathematical programming methods in Python.

The goal is to build a structured computational optimization portfolio combining:

- Mathematical modeling
- Algorithmic implementation
- Benchmark testing
- Statistical performance analysis

---

## 🎯 Objectives

- Implement classical OR problems from the literature
- Solve benchmark instances
- Compare solver performance
- Provide clear and structured mathematical formulations
- Build a practical optimization portfolio

---

## 🛠 Tools and Technologies

- Python
- PuLP
- OR-Tools
- Pandas
- Matplotlib
- SciPy

## 📊 Methodology

For each problem, the following steps are performed:

1. Mathematical formulation
2. Model implementation
3. Benchmark execution
4. Performance data collection
5. Statistical comparison
6. Result visualization
